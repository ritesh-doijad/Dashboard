## Run Locally

Clone the project

```bash
 git clone https://github.com/ritesh-doijad/Dashboard.git
```

Go to the project directory

```bash
cd Dashboard     
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
